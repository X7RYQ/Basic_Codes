About HTML

HTML is the standard markup language for Web pages.

Tags in HTML

There are multiples tags, such as

  1. h1 for the first heading.
  2. p for writing paragraphs or texts.
Few Tags in HTML
S. No. Tag Name Use Of Tag
1. h To write a heading
2. p To write a paragraph.
3. i To write a text in italics.
4. b To write a text in bold.
5. img To display an image.
